System Junk

Takes care of all unneeded data generated by your system and applications.

Note: CleanMyMac includes a built-in Safety Database that ensures none of your vital items are selected for cleanup by default.

Find all system junk on your Mac: Click Scan, and CleanMyMac will accurately analyze all your files and select those that can be seamlessly erased.

Note: If you don’t want to scan some System Junk submodules, go to CleanMyMac 3 > Preferences > Smart Cleanup tab and deselect the appropriate checkboxes. Also note that your choice affects the Smart Cleanup module. For example, if you deselect the User Cache Files submodule, it will be ignored during the scan in both System Junk and Smart Cleanup. If you deselect a checkbox of System Junk (including all submodules), the module will be ignored by Smart Cleanup but fully scanned when you start it separately.

Take a closer look at junk files: Click Review Details. System Junk offers to remove the following file types:

Broken Login Items When you remove an application or service, a broken link to it may remain among your Login Items. CleanMyMac finds and removes it.
Broken Preferences Sometimes application preference files get corrupted, which leads to improper behavior or inability to launch applications. Generally, removing broken preference files eliminates these negative effects.
Document Versions Many apps save versions of a document so that you could restore its earlier versions. It happens automatically, as you work on a doc, or manually, when you command to save changes. The feature is useful but you may be surprised to discover that copies of your documents occupy gigabytes of space. By default, CleanMyMac never offers to remove the earliest and latest version of your doc. It automatically selects only those versions that were made with less than 4-hour interval, keeping the latest one in the range.
Development Junk Applications may create some supporting files during installation and configuration process. They become useless afterward.
iOS Photo Cache After you sync Photos with any of your iOS devices, some photos stay cached on your Mac. This data only wastes space after the sync is done.
Language Files Almost every application comes with multiple localizations you do not use. CleanMyMac removes them to save a significant amount of disk space.
Leftovers Uninstalled applications sometimes leave a lot of junk that occupies space on your Mac but brings no value. CleanMyMac doesn't offer these files for removal by default, so you need to select them manually for cleanup.
macOS Localizations Your Mac's operating system includes dozens of language files, most of which you'll never use. CleanMyMac deletes unneeded language files to save your disk space.
Old Updates Some applications don't clean old versions of their updates. Therefore, these outdated files may take up a lot of space on your Mac. Removing old application updates is safe.
System Cache Files System applications generate and keep many cache files, which results in longer startup time, performance degradation, and possible improper system functioning. You can clean cache and improve performance with CleanMyMac.
System Log Files Activity of your system applications and services is being logged into numerous files. Generally, they are useful only for debugging purposes and, therefore, are hardly worth employing your disk space.
Universal Binaries Developers usually compile their applications to support several Mac architectures, which means putting in twice and sometimes four times the code. However, your Mac only needs one copy.
User Cache Files Originally intended to improve startup time, cache files of your applications ultimately accumulate and result in improper system functioning or overall performance drop.
User Log Files The activity of your applications is being logged into numerous files, which start using a lot of space after a while. However, these files are rarely used and have no value if old.
Xcode Junk If you are a developer, you might want to clean up your system from a significant amount of intermediate build information and project index generated by Xcode. This CleanMyMac ability will come useful when you need to free some space. Note that CleanMyMac doesn't remove Module Cache and Archives automatically.

  • View a file in Finder: Hover over a file you want to reveal and click a magnifier icon next to it.
  • Sort files according to your preference: Click the “Sort by” menu and choose an appropriate option.

Modify the System Junk cleanup list

CleanMyMac automatically adds most of the System Junk items to the cleanup list, but you can prevent removal of any file by simply deselecting its checkbox.

  • Exclude all items from the removal list: Click Deselect All in the upper-left corner.
  • Exclude one or multiple items from the removal list: Deselect checkboxes next to groups or items you want to keep.

Reset scan results: If you want to discard current scan results and start a new scan quickly, choose Action > Rescan. Note that your current scan results and the list of selected items won’t be saved.

Search within a scan results list: Start typing a word in the search field - results appear below the field right as you type. To view all available items, click the close button (x).

Add files to CleanMyMac’s Ignore List: If CleanMyMac’s scan results contain a file you do not want to be ever removed, control-click the corresponding file and choose Add to Ignore List from the shortcut menu.

Click Clean to remove junk and improve your Mac’s performance.

Use the Start Over button to reset scan results and go back to the initial System Junk screen.
