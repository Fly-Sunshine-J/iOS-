Trash Bins

Empties all Trash folders on your Mac, including Photos trash, external drives trash, and many more.

Apart from ordinary startup drive Trash, Macs generally have additional application trash folders, for example, Photos Trash. Each disk partition and external drive also has its hidden trash folder where previously deleted items reside.

The Trash Bins module the following folders:

  • Local Mail Trash, which contains emails deleted in the Mail application.
  • Aperture Trash, which contains photos deleted in Aperture.
  • iPhoto Trash, which contains photos deleted in iPhoto.
  • Photos Trash, which contains images deleted in Photos.
  • Gemini Duplicates, which contains duplicated images that you have earlier removed from a Photos library using Gemini.
  • Startup Drive Trash, which contains items deleted from the main disk and items removed from iCloud.
  • External Drive Trash, which contains items deleted from external drives (the number of such Trash folders equals the number of external drives).

Find out how much space is occupied by the files you’ve already deleted: Click Scan to analyze all your trash folders.

Take advantage of Smart Selection. Click Empty to remove automatically selected items. CleanMyMac adds all trashed items to the cleanup list automatically, except for Gemini Duplicates, which can be removed via the Photos app only. Also, note that starting from macOS 10.13, Photos Trash can be removed via the Photos app only.

Review the content of trash folders: Click Review Files.

  • View a file in Finder: Hover over a file you want to reveal and click a magnifier icon next to it.
  • Take a look at a file in CleanMyMac: Hover over a file you want to reveal and click a Quick Look icon next to it.
  • Sort results: Click the “Sort by” menu and choose an appropriate option.

Reset scan results: If you want to discard current scan results and start a new scan quickly, choose Action > Rescan. Note that your current scan results and the list of selected items won’t be saved.

Search within a scan results list: Start typing a word in the search field - results appear below the field right as you type. To view all available items, click the close button (x).

Change the Trash Bins cleanup list

CleanMyMac smart-selects all trashed items in all trash bins on your Mac. If you want to keep some of them, deselect their checkboxes.

  • Exclude all items from the removal list: Click Deselect All.
  • Exclude one or multiple items from the removal list: Deselect checkboxes next to trash folders or items you want to keep.
  • Add one or multiple items to the removal list: Select checkboxes next to trash folders or items you want to remove.
  • Add all items to the removal list: Click Select All.

Clean up your Mac from trash

  • Remove immediately: Click Empty to clean trash folders quickly. This is a default method that removes files from your disk in an ordinary way.
  • Remove securely: After scan is over, click Review Files. Then, choose Remove Securely from the pop-up menu next to the Empty button. Click Empty to remove all selected item in a secure way. This takes more time, but CleanMyMac rewrites all data with zeroes and, therefore, makes the recovery impossible. You can be sure that deleted items will not be recovered by means of any special tools.

Note: Despite the fact that the Secure Empty Trash feature is missing in macOS El Capitan (10.11 or higher), CleanMyMac still effectively removes your data in a secure way.

Remove Gemini Duplicates (only for the Gemini app users)

Open Photos and navigate to the Gemini Duplicates album. Then, select and control-click all contents of the album. From the shortcut menu, choose Delete Photos/Videos.

Remove Photos Trash

Starting from macOS 10.13, Photos files can be removed via Photos app only. If you have an earlier version of macOS, CleanMyMac easily removes these files without additional actions.

So, to delete Photos library trash on macOS 10.13 and later, open Photos and navigate to the Recently Deleted album. Then, click Delete All in the upper right corner of the Photos window.
