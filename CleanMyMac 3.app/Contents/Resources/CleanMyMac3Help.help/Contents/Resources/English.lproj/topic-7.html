Large & Old Files

Finds large and possibly forgotten files or folders you might want to remove.

Change folders for analysis

You can either use a default location, your Home folder, or select any other place for analysis. It can be an internal or external drive, network folder, or any other location.

  • Select a new location: Click Add Folder and navigate to the location you want to scan.
  • Exclude a location from the scan list: Click the close button (x) next to the location you do not want to analyze.
  • Exclude all locations from the scan list: Click Clear List.

Find large and forgotten files: Click Scan to analyze the selected locations.

Review the results: Click Review Files.

  • View a file in Finder: Hover over a file you want to reveal and click a magnifier icon next to it.
  • Take a look at a file in CleanMyMac: Hover over a file you want to reveal and click a Quick Look icon next to it.
  • Sort results: Click the “Sort by” menu and choose an appropriate option.

Select files for removal

CleanMyMac does not smart-select any files found in the Large & Old Files module.

  • Select files for removal: Select checkboxes next to files you want to remove.
  • Exclude files from the removal list: Deselect checkboxes next to files you want to keep on your Mac.

Add files to the Ignore List: If CleanMyMac’s scan results contain a file you do not want to be ever removed, control-click the corresponding file and choose Add to Ignore List from the shortcut menu.

Reset scan results: If you want to discard current scan results and start a new scan quickly, choose Action > Rescan. Note that your current scan results and the list of selected items won’t be saved.

Search within a scan results list: Start typing a word in the search field - results appear below the field right as you type. To view all available items, click the close button (x).

Put files to a more appropriate place or remove them to free up space

  • Move files to another folder: Choose Move to Folder from the pop-up menu next to the Remove button and navigate to a folder where you want to place your files.
  • Move files to the Trash folder: Choose Move to Trash from the pop-up menu next to the Remove/Move button if you want to place a file to the Trash folder and decide later on its value.
  • Remove immediately: Choose Remove Immediately from the pop-up menu next to the Remove/Move button and click Remove.
  • Remove securely: Choose Remove Securely from the pop-up menu next to the Remove/Move button. Click Remove to delete all selected item in a secure way. This takes more time, but CleanMyMac rewrites all data with zeroes and, therefore, makes the recovery impossible. You can be sure that deleted items will not be recovered by means of any special tools.

Note: Despite the fact that the Secure Empty Trash feature is missing in macOS El Capitan (10.11 or higher), CleanMyMac still effectively removes your data in a secure way.

Remove Photos files

Starting from macOS 10.13, Photos files can be removed via Photos app only. If you have an earlier version of macOS, CleanMyMac easily removes these files without additional actions. So, to delete large and old Photos files on macOS 10.13 and later, do the following:

  1. Find items you’d like to remove in CleanMyMac’s Large & Old Files and select them.
  2. Click Remove.
  3. Open Photos and navigate to the Large Media to Delete album.
  4. Select all contents of the album.
  5. Control-click any selected item and choose Delete Photos/Videos from the shortcut menu.

Now, your Photos library is clean.
