Uninstaller

Correctly removes applications with all of their related files or resets them to the default state.

Uninstaller shows you all custom applications available on your Mac, including those that aren’t stored in the Applications folder.

Note: Uninstaller does not support the removal of obligatory system applications and does not display them by default.
However, you can change CleanMyMac preference to make it show this type of apps. When system apps become visible, you can reset them.
Take the following steps to make system application visible:

  1. Go to CleanMyMac 3 > Preferences > Ignore List > Uninstaller.
  2. Deselect the Ignore system applications checkbox.

Using information from Uninstaller, you can:

  • Reset an application to the initial state to resolve some troubles or to make it run faster.
  • Remove any application with all related files, leaving no remains.
  • View unsupported applications.
  • Reset system applications.

Review all your applications: Click View All Applications.

  • View an item in Finder: Hover over a file you want to reveal and click a magnifier icon next to it.
  • View an item in CleanMyMac: Hover over a file you want to reveal and click a Quick Look icon next to it.
  • Sort results: Click the “Sort by” menu and choose an appropriate option.
  • View rarely used applications: Choose “Sort by” > Last Launch Date to find applications that make little use for you.

Refresh current Uninstaller results: If you want to update the current list of applications, choose Action > Refresh. Note that the current selection of items won’t be saved.

Search within a search results list: Start typing a word in the search field - results appear below the field right as you type. To view all available items, click the close button (x).

Select applications you want to remove or reset

Uninstaller does not offer you to remove anything. You must decide what you would like to do with each application:

  • Add an application for uninstallation: Select a checkbox next to an application you no longer need, and then click Complete Uninstallation in the Smart Selector block.
    Note that you cannot uninstall a system application.
  • Add an application for reset: Select a checkbox next to an application you want to reset, and then click Application Reset in the Smart Selector block. This option automatically selects items that must be removed to reset the application to its initial state. The application starts working with a clean state when you launch it next time.

To exclude an application or file from the removal list, deselect the checkbox next to it.

Add applications to the Ignore List: If CleanMyMac’s scan results contain an application you do not want to be ever removed, control-click the corresponding file and choose Add to Ignore List from the shortcut menu.

Uninstall/reset the selected applications: Depending on the selected items, the main CleanMyMac button identifies the operation you are about to start:

  • Uninstall: Click this button to remove all selected applications leaving no remains on your Mac.
  • Reset: Click this button to remove smart-selected files, and thus reset the selected applications.
  • Remove: Click this button to uninstall selected applications and/or remove files of your choice.

As soon as any operation is done, use the View All Apps button to return to the list of your applications.
