Dashboard

Shows a current state of system resources and allows you to clean up your RAM.

The Dashboard displays real-time state infographics of your internal drives, random-access memory (RAM or memory), battery, and processor (CPU).

CleanMyMac is constantly monitoring the system and lets you know if there are any health issues. However, you might want to open the Dashboard periodically to improve your Mac’s performance through RAM cleanup or to check available disk space, processor workload, and battery state.

Assess your Mac’s current state at a glance

  • Open the Dashboard: Click Dashboard in CleanMyMac’s upper-right corner.

The Dashboard displays your Mac’s model, total amount of cleaned data, and detailed information on your Mac’s components represented in a form of graphs.
A graph is a visual representation of one of the components containing a legend of all of its items.

  • Check a component’s state: Hover over a graph to reveal its legend.
Hard drives

CleanMyMac monitors all internal volumes of your Mac and displays them individually with their icons, names, and graphs.

Each graph contains color-encoded sections that represent different categories of files stored on a disk. The size of a sector correlates with the amount of disk space used by a file category it represents and is specified in the legend.

You can see the following file categories on a disk graph:

  • Audio - your iTunes Library music and files in MP3, AAC, AIFF, APE, and other audio formats.
  • Movies - video files, including movies and podcasts of all known video formats.
  • Photos - your iPhoto Library images and files in JPEG, PNG, TIFF, and other image formats.
  • Apps - application files from any location on the drive.
  • Backups - archives, containers, and backup files of different origin.
  • Other - all files that cannot be referred to any of the listed above file categories. For example, contacts, calendar data, the System folder, cache data, application plugins, and so on.
  • Used - a single category that shows the amount of occupied place if, for some reason, categorization is not available.

The amount of available disk space is displayed right above the hard drive graph. It represents the total amount of both free and purgeable space that can be instantly released by the system. Learn more about purgeable space here.

Take a closer look at a disk or at a file category

  • View a disk in Finder: Click on a disk icon.
  • Open a Finder window with the Spotlight search for a particular file category: Double-click the corresponding section of the graph.
  • View a short file category description in the legend: Click a category name.
Memory

RAM keeps the information that is being processed by your system and applications. Generally, it helps speed up completion of operations, but as the amount of data stored in the memory grows, your Mac may become slower.

Clean up the memory to make it run faster: Click Free Up. Be sure, this operation does not affect any of your files located on hard drives.

As you hover over the Memory graph, you see the following data categories:

  • Wired - information in RAM that can't be moved to the drive or used by other applications. The amount of wired memory depends on the system configuration and applications you are using.
  • Active - information that is currently being used and constantly addressed to. 
  • Inactive - information that was recently used but remains currently unaddressed.
  • Compressed - the amount of memory used by inactive applications that was compressed to make more RAM available to active applications.

Take a closer look at the memory: Click on the Memory icon to open Activity Monitor.
Battery

Check the battery status: The current charge level of the battery is displayed above the Battery graph.

View details of the battery state: Hover over the Battery graph to check its health and a number of charge cycles.

Take a closer look at the battery: Click on the Battery icon to open Energy Saver Preferences (in System Preferences).
Processor

This Dashboard element shows the current CPU (processor) workload in %.

Take a closer look at the processor's current state: Click on the Processor icon to open Activity Monitor.

Manage Mac’s health issues

If CleanMyMac notices some issues that may affect your Mac’s performance, it changes the Dashboard button title to warn you. For example, if CleanMyMac detects two issues that directly affect system processes, you’ll see “2 issues” button instead of the “Dashboard.”

  • Check a health issue: Open the Dashboard and click on a yellow warning triangle next to a component. CleanMyMac will show a pop-up message with a description of the health issue and applicable recommendations.

Note: The warning icon does not disappear from the Dashboard until the issue is resolved.
