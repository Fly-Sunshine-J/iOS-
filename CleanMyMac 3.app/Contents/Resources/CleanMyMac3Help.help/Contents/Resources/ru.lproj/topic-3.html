Фотомусор

Удаляет вспомогательные файлы программы «Фото» без вреда для фотографий. Заменяет исходные изображения в «тяжелом» формате RAW их высококачественными JPEG-копиями.

В модуле «Фотомусор» CleanMyMac сканирует медиатеку программы «Фото» в поисках следующих типов данных:

Оригиналы RAW

(не поддерживается в macOS 10.13+)

Большинство камер сохраняет фотографии в RAW-формате, который не используется в обычных редакторах и программах для просмотра изображений. Поэтому когда Вы импортируете фотографии в программу «Фото», она создает визуально идентичные копии изображений в JPEG-формате, которые Вы и видете в программе.

CleanMyMac находит RAW-оригиналы фотографий и заменяет их JPEG-изображениями высокого качества, что позволяет сохранить много места.
Кэш Лиц В процесе работы с функцией «Лица» создается много вспомогательных файлов. Они быстро устревают или перестают использоваться программой, поэтому их удаление никак не влияет на работу «Фото».
Фото кэш Программа «Фото» создает несколько типов файлов кэша, которые абсолютно безопасны для удаления. Она автоматически генерирует их, как только в этом возникает необходимость.
Локальные копии iCloud

Если Вы подключите медиатеку iCloud к программе «Фото», на Ваш Мас будут сохраняться копии всех просмотренных фотографий. CleanMyMac позволяет удалить эти копии без вреда для изображений в медиатеке iCloud.

Помните, если Вы решите удалить локальные копии, Вам понадобится подключение к Интернету, чтобы снова просмотреть эти фотографии.

Выбор медиатеки «Фото» для анализа. Вы можете оставить для анализа текущую медиатеку или выбрать другую с помощью кнопки «Сменить медиатеку».

Поиск мусора в медиатеке «Фото». Нажмите кнопку «Старт», и CleanMyMac найдет все объекты, которые можно безопасно удалить с Вашего Мас.

Быстрая очистка. CleanMyMac автоматически выбирает для удаления только объекты из групп «Фото кэш» и «Кэш Лиц». Нажмите кнопку «Очистить», чтобы удалить фотомусор.

Просмор подробных результатов анализа медиатеки. Нажмите кнопку «Подробнее».

Обзор оригиналов RAW

  • Масштабирование. Перетаскивайте ползунок в правом верхнем углу для масштабирования изображения.
  • Просмотр изображений в программе «Фото». Удерживая клавишу Control, нажмите изображение, затем выберите «Показать в Фото».
  • Сравнивание RAW-оригинала и JPEG-версии изображения. Удерживая клавишу Control, нажмите изображение, затем выберите «Быстрый просмотр» для просмотра изображений бок о бок.

Сортировка и изменение списка к удалению

  • Изменение сортировки найденных объектов. Нажмите меню «Сортировать по» вверху окна, и выберите подходящий критерий сортировки.
  • Выбор отдельных изображений или групп для удаления. Установите флажки для групп или изображений, которые Вы хотите удалить.
  • Выбор всех изображений для удаления. В левом верхнем углу нажмите кнопку «Выбрать все».
  • Исключение всех изображений из списка к удалению. В левом верхнем углу нажмите кнопку «Отменить все».
  • Исключение отдельных изображений или групп из списка к удалению. Снимите флажки для групп или изображений, которые Вы хотите оставить.

Обновление результатов сканирования. Если Вы хотите сбросить результаты поиска и быстро начать новое сканирование, выберите «Действие» > «Повторная проверка». Обратите внимание, текущие результаты и список выбранных файлов не сохранятся.

Поиск по списку. Начните вводить слово в поле поиска - результаты будут появляться по мере ввода текста. Чтобы просмотреть все доступные объекты, нажмите значок «Закрыть» (х).

Добавление изображений в список исключений. Если Вы уверены, что никогда не захотите удалить определенную фотографию, выберите из ее контекстного меню «Добавить в список исключений».

Примечание: По умолчанию CleanMyMac не добавляет RAW-фотографии в список удаления. Тем не менее, Вы можете вообще исключить из поиска CleanMyMac группу «Оригиналы RAW». Для этого выберите CleanMyMac 3 > «Настройки» > «Список исключений». В разделе «Фотомусор» установите флажок «Игнорировать RAW фотографии».

Нажмите кнопку «Очистить», чтобы удалить ненужные данные из Вашей медиатеки.

Кнопка «Начать заново» вернет Вас на стартовый экран модуля.

Используете медиатеки iPhoto?

CleanMyMac находит все скрытые оригиналы изображений в пределах выбранной медиатеки iPhoto. В частности, копии фотографий, сохраненные в формате RAW или отредактированные другими способами:

  • Обрезанные
  • Повернутые
  • С коррекцией красных глаз
  • Улучшенные

Примечание: По умолчанию CleanMyMac выбирает только оригиналы фотографий после исправления эффекта красных глаз и исправления ориентации.

На экране с подробными результатами сканирования CleanMyMac отображает изображения попарно – скрытый оригинал и окончательная версия со всеми изменениями. Наведите курсор на изображение, чтобы увидеть подсказки.

Для Вашего удобства изображения можно группировать по следующим критериям: модификаторы, события или дата.
