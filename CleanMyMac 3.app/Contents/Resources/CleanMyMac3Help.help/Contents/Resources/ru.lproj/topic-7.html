Большие файлы

Находит большие и забытые файлы, которые, возможно, Вы хотели бы удалить.

Выбор папки для проверки

По умолчанию CleanMyMac предлагает проанализировать Вашу домашнюю папку, но Вы можете выбрать любую другую папку на внутреннем, внешнем или сетевом диске.

  • Добавление новой папки. Нажмите кнопку «Добавить папку», и перейдите к папке, которую Вы хотите проверить.
  • Исключение отдельной папки из списка сканирования. Нажмите значок «Закрыть» (х) напротив папки, которую Вы не хотите проверять.
  • Исключение всех папок из списка сканирования. Нажмите кнопку «Очистить список».

Поиск больших и забытых файлов. Нажмите кнопку «Старт» для анализа содержимого выбранных папках.

Обзор результатов. Нажмите кнопку «Обзор файлов».

  • Просмотр объекта в Finder. Наведите курсор на файл, и нажмите значок с изображением увеличительного стекла.
  • Быстрый предварительный просмотр. Наведите курсор мыши на файл, который Вы хотите просмотреть, и нажмите значок быстрого просмотра.
  • Сортировка результатов. Нажмите меню «Сортировать по» в верхней правой части окна, и выберите подходящий критерий сортировки.
  • Группировка файлов. Используйте меню в верхней левой части окна, чтобы выбрать подходящий критерий группировки найденных объектов:
    • по размеру (50 МБ - 500 МБ, 500 МБ - 1 ГБ и т. д.);
    • по дате последнего открытия (сегодня, вчера, месяц назад и т. д.);
    • по типу данных (фильмы, папки, архивы и т. д.).

Выбор файлов для удаления.

CleanMyMac никогда не предлагает к удалению файлы, найденные в этом модуле.

  • Исключение файлов из списка к удалению. Снимите флажки для групп или объектов, которые Вы хотите оставить.
  • Выбор файлов для удаления. Установите флажки для файлов, которые Вы хотите удалить.

Добавление объектов в список исключений CleanMyMac. Если Вы уверены, что никогда не захотите удалить определенный объект, выберите из его контекстного меню «Добавить в список исключений».

Обновление результатов сканирования. Если Вы хотите сбросить результаты поиска и быстро начать новое сканирование, выберите «Действие» > «Повторная проверка». Обратите внимание, текущие результаты и список выбранных файлов не сохранятся.

Поиск по списку. Начните вводить слово в поле поиска - результаты будут появляться по мере ввода текста. Чтобы просмотреть все доступные объекты, нажмите значок «Закрыть» (х).

Перемещение файлов в более подходящее место или их удаление для экономии места

  • Перемещение файлов в другую папку. Выберите из меню опцию «Переместить в папку» и перейдите к папке, куда необходимо переместить файлы.
  • Перемещение файлов в корзину. Выберите из меню опцию «Переместить в корзину», если Вы хотите переместить файл в корзину и определить его судьбу позже.
  • Простое удаление. Чтобы удалить файлы незамедлительно, выберите «Просто» из меню справа от кнопки «Удалить/Переместить». Затем нажмите кнопку «Удалить».
  • Удаление с перезаписью: Чтобы исключить возможность восстановления удаленных объектов специальными программами, выберите опцию «С перезаписью» из меню справа от кнопки «Удалить/Переместить». Затем нажмите «Удалить». Обратите внимание: удаление файлов с перезаписью занимает больше времени, нежели простое удаление.

Примечание: Несмотря на то что в macOS El Capitan (версии 10.11 и выше) отсутствует функция «Очищать Корзину необратимо», в CleanMyMac по-прежнему можно удалить файлы с перезаписью (без возможности восстановления).

Удаление файлов программы «Фото»

Начиная с macOS 10.13, файлы программы «Фото» можно удалить только непосредственно из этой программы. Однако если у Вас установлена более ранняя версия macOS, CleanMyMac легко удалит данные файлы самостоятельно. Чтобы удалить большие или устаревшие файлы из программы «Фото», сделайте следующее:

  1. В модуле «Большие файлы» найдите и выделите объекты, которые Вы хотите удалить.
  2. Нажмите «Удалить».
  3. Откройте программу «Фото» и перейдите в альбом «Файлы к удалению».
  4. Выберите все содержимое альбома.
  5. Щелкните правой кнопкой мыши по любому выделенному объекту и выберите «Удалить фото/видео» из контекстного меню.

Все, теперь Ваша медиатека очищена.
