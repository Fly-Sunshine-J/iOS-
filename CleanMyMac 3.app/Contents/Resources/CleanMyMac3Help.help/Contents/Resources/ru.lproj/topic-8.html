Деинсталлятор

Корректно удаляет программы и их родственные файлы либо возвращает настройки по умолчанию.

В модуле «Деинсталлятор» CleanMyMac собирает все установленные на Вашем Mac программы, в том числе те, которые не находятся в папке «Программы».

Примечание: «Деинсталлятор» не удаляет предустановленные системные программы и не отображает их в списке доступных приложений по умолчанию.
Вы можете включить отображение системных программ в «Деинсталляторе», изменив настройки CleanMyMac. Для этого выберите CleanMyMac 3 > «Настройки» > «Список исключений». В разделе «Деинсталлятор» снимите флажок «Игнорировать системные программы».

С помощью «Деинсталлятора» Вы можете:

  • обнулить программу до исходного состояния, чтобы ускорить ее или устранить ошибки в работе;
  • удалить любую программу со всемми вспомогательными файлами;
  • узнать, какие программы Ваш Мас более не поддерживает;
  • обнулить системную программу.

Просмотр доступных программ. Нажмите кнопку «Просмотреть все программы».

  • Просмотр объекта в Finder. Наведите курсор на файл, и нажмите значок с изображением увеличительного стекла.
  • Сортировка результатов. Нажмите меню «Сортировать по» в верхней части окна, и выберите подходящий критерий сортировки.
  • Просмотрите редкоиспользуемые программы: Из меню «Сортировать по» выберите «Дата запуска». CleanMyMac объединит программы в группы по дате последнего запуска (например, «Месяц назад»). Refresh current Uninstaller results: If you want to update the current list of applications, choose Action > Refresh. Note that the current selection of items won’t be saved.

Обновление списка файлов «Деинсталлятора». Если Вы хотите обновить текущий список программ и файлов, выберите «Действие» > «Обновить». Обратите внимание, список выбранных файлов не сохранится.

Поиск по списку. Начните вводить слово в поле поиска - результаты будут появляться по мере ввода текста. Чтобы просмотреть все доступные объекты, нажмите значок «Закрыть» (х).

Выбор программы для удаления или обнуления

По умолчанию «Деинсталлятор» никогда не предлагает Вам удалить или обнулить программы. Вы должно самостоятельно сделать выбор для каждой программы:

  • Удаление программы. Установите флажок для программ, которые Вы хотите удалить. Или выделите программу в списке, а затем нажмите кнопку «Полное удаление» из меню Быстрого выбора.
    Обратите внимание: системную программу невозможно удалить полностью, ее можно только обнулить.
  • Обнуление программы. В списке выделите программу, которую Вы хотите обнулить, и нажмите кнопку «Сброс настроек программы» из меню Быстрого выбора. Эта функция автоматически выбирает объекты, которые необходимо удалить для возвращения исходных настроек программы. После обнуления и перезапуска программа откроется как свежеустановленная.

Чтобы исключить программу или файл из списка удаления, снимите флажок соответствующего объекта.

Добавление программ в список исключений CleanMyMac. Если Вы уверены, что никогда не захотите удалить или сбросить настройки определенной программы, выберите из ее контекстного меню «Добавить в список исключений».

Удаление или обнуление выбранных программ. В зависимости от выбранных объектов, основная кнопка CleanMyMac изменит свое название на одно из следующих:

  • Удалить. Нажмите кнопку для удаления всех выбранных программ и объектов.
  • Обнулить. Нажмите эту кнопку, чтобы удалить автоматически выбранные файлы и вернуть первоначальное состояние программ.

Нажмите кнопку «Просмотр всех программ» после выполнения любой из операций чтобы вернуться к списку программ и продолжить работу с модулем.
