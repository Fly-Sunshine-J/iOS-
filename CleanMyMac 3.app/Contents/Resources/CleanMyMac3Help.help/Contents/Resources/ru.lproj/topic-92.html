Шредер

Бесследно стирает любые ненужные файлы или папки.

«Шредер» легко удаляет ненужные файлы как обычным, так и защищенным способом. Он без труда справляется с задачей, даже если Finder возвращает ошибки при попытке удалить файлы, вовлеченные в текущие процессы.

Выбор файлов для удаления. Нажмите кнопку «Выбрать файлы», найдите объект, который Вы хотите удалить и нажмите кнопку «Открыть».

Изменение списка к удалению

  • Исключение ранее выбранных объектов из списка к удалению. Снимите флажки для объектов, которые Вы хотите оставить. Чтобы исключить все объекты из списка удаления, нажмите кнопку «Отменить все» в левом верхнем углу.
  • Добавление новых объектов для удаления. Нажмите кнопку «Добавить файлы» и выберите новые объекты для удаления.

Изменение критерия сортировки. Нажмите меню «Сортировать по» в верхней правой части окна и выберите подходящий критерий сортировки.

Поиск по списку. Начните вводить слово в поле поиска - результаты будут появляться по мере ввода текста. Чтобы просмотреть все доступные объекты, нажмите значок «Закрыть» (х).

Удаление выбранных файлов

  • Простое удаление. Чтобы удалить файлы незамедлительно, выберите «Просто» из меню справа от кнопки «Стереть». Затем нажмите кнопку «Стереть».
  • Удаление с перезаписью. Чтобы исключить возможность восстановления удаленных объектов специальными программами, выберите опцию «С перезаписью» из меню справа от кнопки «Стереть». Затем нажмите «Стереть». Обратите внимание: удаление файлов с перезаписью занимает больше времени, нежели простое удаление.

Примечание: Несмотря на то что в macOS El Capitan (версии 10.11 и выше) отсутствует функция «Очищать Корзину необратимо», в CleanMyMac по-прежнему можно удалить файлы с перезаписью (без возможности восстановления).

Кнопка «Начать заново» вернет Вас на стартовый экран модуля.
